# Inspired

Inspired by https://github.com/holman/dotfiles

# Install

Clone into home folder

`git clone ~/.dotfiles`

# Run 

Run bootstrap script

`cd ~/.dotfiles`

`./bootstrap`

# To test

Run `vagrant up` to provision an osx virtual machine to test this in.
